windows 无法安装到这个磁盘。选中的磁盘采用的是 GPT 分区形式

现在的系统大都采用的是优先 MBR（主引导记录）引导的系统，而部分电脑则采用的是优先 EFI（可扩展固件接口，一种先进的系统引导方式）引导的系统。一般情况下，现今的电脑系统同时支持以上两种引导方式。当进行系统重装时，用前者优先引导的系统总是不能装到后者引导的硬盘上，总是出现错误提示：“windows 无法安装到这个磁盘。选中的磁盘采用的是 GPT 分区形式。”

按【shift+F10】调出 diskpart 命令提示符，
然后依次输入diskpart （调出磁盘管理工具）
list disk （列出所有盘符）
select disk 0 （一般情况下，disk 0 为系统硬盘）
clean （清除系统硬盘和非系统盘所有数据）
convert mbr （转换硬盘引导方式为 MBR）
create partition primary size = xxx （创建主分区，单位M） （对硬盘重新分区，如果不输入，可在系统安装完成后，在系统的磁盘管理里再进行分区）
format fs=ntfs quick （转换硬盘格式为 ntfs 格式，压缩方式为 quick ）exit （退出磁盘分区设置）exit （退出磁盘转换设置）
刷新磁盘列表
之后，就可以正常安装了
